# Projeto Prático de Programação para Dispositivos Móveis 1

## Objetivo do Aplicativo
Este aplicativo tem como objetivo calcular a distância entre a localização atual do usuário e um local determinado.  
Para isso, utiliza o recurso de localização por meio da biblioteca expo-location, que permite capturar as coordenadas geográficas atuais do dispositivo.  
Com base nas coordenadas obtidas, o aplicativo realiza o cálculo da distância até o destino informado, oferecendo uma forma simples e prática de acompanhar deslocamentos.  
Além disso, os destinos são salvos em banco de dados local utilizando SQLite, garantindo persistência mesmo após fechar o app.
